Project overwiev

Project was the university class exam. The goal was to make PHP CMS website that will feature multi user/role platform. The user sign in persistance trough session, clearning session variable on its end, hashing of password. Validation and sanitisation of any user input as well as use of prepared SQL statements. Inform user on any erros and file upload , pagination of content,separation of files and ajax content load .

The author idea was to make an learning platfrom like website where user can enrolll in class. Class would consist of pages featuring diffrent content type but with fixed design.

Project implements all the features demanded by project goal list,except for active JS validation (during user input), while being super modular trough OOP. User can register and login,create classes,edit them,choose different content type img,video,yt video . It can also edit his account as well as delete it.Admin can manage users. App uses session based atuhtentification , parametric SQL and hashes passwords.
