Digital circuit sim

Project overwiev

This is an small project idea i built during university. Goal was to make digital electronics simulator.The project manages to simulate logic gates accuratly. Only limitations are lack of editing tools,wires are just drawn and limited workspace.

DEMO

How simulation works

Once user has placed components it hits Run button. During run swing worker runs infinite loop which calls wire updates. All components are child of digitalComponentClass which has update method. Update method takes input values and sets output value and each component has its own ovrrride. On other hand wires take output values and put them on input. Aditonally wires call component update esentaily propagating states trough circuit.

More implementation details

Componets consist of visual and simulator part. In reality its just JPanel with texture while pins are Jbuttons. Pins use txt file to setup pin positions within component. Components are made on click of component button using component factory (generics and reflection applied) . Code is even designed for quick adition of new components trough use of texture file + component class name enum + dig component inheritance.
