Java Digital simulator

This is a litle project i decided to build while in university. I already was comftorable with Swing and Digital electronics are just logic so why not.

Originaly it was motivated by idea that i might do some digital electronics with redstone in minecraft in hope of building 8 bit computer .But designing scheme is hard and not interactive so i kinda got idea to build a simulator in java.

The project works on realtivly simple logic, all components use same class digital component which contains method update . Update method updates output pins based on input ones while wires transfer state from input to output. Project builds the buttons for components dinamycally. It also creates components dinamycally as whole app was designed to feature easy addition of new components. Component in reality is just an JPanel with texture while pins are buttons, to offset btns and match texture tehre is text file which stores pins offsets.

Overall while the project manages to correctly simulate the circuits in swing worker , the movement and wires proved to be difficult to implement so in end project was abandoned. The movement is only most limiting factor in this program as it limits workspace and no abiltiy to move compoents.
