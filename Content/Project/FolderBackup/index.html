Folder Backuper

Project overview

During november i heard father needs backuping solution. At the time i was finished with mcWorldMangerV1 so i taken the stuff i practiced there to make program for this specific case.

Goal was to make program which would automatically backup list of folders containting files to desired location.

The backuper was succefully made and makes folder copies .It can operate in interval or backup on startup mode. Also it can minimise to tray. However few issues remain ,main one including use of recursion during copying. This could potentially cause overflow ,plus it runs on UI thread so app can freeze.
