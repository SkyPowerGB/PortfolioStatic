Minecraft world manager

Project motivation

Minecraft Saves folder is located inside appData roaming folder. Issue is that its vournable to main drive failure while manual backuping is a bit annyoing and can be easily forgotten.

For that reason i decided to make this program that will do just that ,make backuping quick but i also included some other features.

Project Overview

mcWorldManager is program that allows user to manage his minecraft worlds. It allows for storage of worlds on other drive , the backups must include metadata file for aditional info. It also has QuickBackup option for one click backup of saves folder,stored seperatly using datestamp. Quick backup has automatic interval run mode.

Aditional features include grouping (branching), for same named worlds. Another great feature is ability to edit NBT data so you can change world name within game as well as edit game mode and cheats. This is quite useful for editing worlds stuck in survival or hardcore. (Example: i had old world i made in survival but plaied in creative, this constatnly caused issue of needing run lan mode to allow cheats). And lastly app has ability to minimise into tray.

Libraries

Program uses .NetFramework for file/folder operations.

The newtonsoftJson is used for persistent storage and metadatafiles.

Fnbt is used for NBT files. NBT stands for "Named Binary Tag" which is binary file tree structured format used by minecraft.
