MC world manager V1

An comprehensive minecraft world manager and backuper program made in win forms. It allows for quick and easy world managment and backuping all while having aditonal features like NBT editing.

Features

The user can move worlds between backup folder and saves folder within appdata/roaming/.minecraft/saves. In order for user to move worlds to backup they must be market which is just json metadata file to contain data about world. Metadata stores world and minecraft version ,description and copy date. Another option app adds is QBackup which is one click backup of saves folder. And finnally user can edit world name, edit some NBT file points like game mode and difficuluty, add "branch" which is just folder and meant for grouping same name worlds with different versions .

Problems it aimed to solve

Problem it was meant to solve is to make making backups quicker and easier. Minecraft worlds are located in appdata roaming so just locating folder is slow, plus its located on C drive which poses risk if main drive gives up and you can loose your worlds.Making this one click makes it simpler. Aditonal features needed were cases of same name world : which is the latest?, which world (in older versions) is what mc version? Aditonal notes ? . Another potential feature was conflicting worlds with uncertain date or bracnhed out ones . And finnaly as nice add on editing NBT in case of failed HC worlds to turn them into creative/survival, missed allow cheats and similar problems.

Conclusion

Program does the job reasonably well albeit UI is not fully polished and some features werent fully built like shearching. It can be used to manage worlds . It can change some world data as needed. It can make backups quick or loading stuff from backups.
